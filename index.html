MINHAS ESPECIALIDADES.

HTML

HTML, abreviação de HyperText Markup Language, é a linguagem fundamental usada na criação de páginas da internet. Ela usa "tags" para marcar diferentes tipos de conteúdo, estruturando textos, imagens, links e outros elementos em páginas web

CSS

CSS (Cascading Style Sheets) é uma linguagem de estilo usada para estilizar páginas web, controlando o layout, cores e aparência visual dos elementos HTML. Essencial para o design web, o CSS permite a criação de páginas atraentes e consistentes.

JAVA SCRIPT

JavaScript é uma linguagem de programação usada para tornar páginas web interativas. Executada no navegador, ela manipula o conteúdo, responde a eventos e aprimora a experiência do usuário.
